## Big Data Basics

Big data pain points: 

- I/O bottleneck
- network bottleneck
- data shuffling 
- processing time


Big data challenges / characteristics - 5 Vs: 

- volume
- velocity
- variety 
- veracity 
- value

Not all organizations experience challenges in every area. 

Key aspects of a big data platform:

- integration 
- analysis
- visualization 
- workload optimization 
- security 
- governance
